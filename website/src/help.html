Proselint

More information and help on proselint, or go back.

Install

Via Pip

pip install proselint

Via Apt (Debian / Ubuntu)

apt install python3-proselint

Via Homebrew (macOS / Linux)

brew install proselint

Via Conda

conda install conda-forge::proselint

Via Nix

environment.systemPackages = with pkgs; [
  proselint
];

From source

git clone https://github.com/amperser/proselint.git
cd proselint

uv pip install . # or, with Nix
nix profile install .

Proselint searches for configuration files in the following order of priority:

  • .proselintrc.json in the current working directory
  • $XDG_CONFIG_HOME/proselint/config.json
  • $HOME/.proselintrc.json
  • /etc/proselintrc

To override a rule locally, create .proselintrc.json. For example: 

{
  "checks.typography.symbols.curly_quotes": false
}

Wire Output

Structured output can be enabled with --output-format json: 

proselint --output-format json syllabus.md

The output schema is:

enum ErrorCode {
  Unknown = -31999,
  FileError = -31998,
  LintError = -31997,
}

interface ResponseError {
  code: ErrorCode;
  message: string;
  data?: any;
}

interface CheckResult {
  check_path: string;
  message: string;
  span: [number, number];
  replacements: string | null;
}

interface LintResult extends CheckResult {
  pos: [number, number];
}

interface FileOutput {
  diagnostics?: LintResult[];
  error?: ResponseError;
}

interface ProselintOutput {
  result?: Record;
  error?: ResponseError;
}

Each invocation produces exactly one ProselintOutput. Both ProselintOutput and FileOutput are tagged unions.
