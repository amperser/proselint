Proselint

More information and help on proselint, or go back.

Install

Via Pip

pip install proselint

Via Apt (Debian / Ubuntu)

apt install python3-proselint

Via Homebrew (macOS / Linux)

brew install proselint

Via Conda

conda install conda-forge::proselint

Via Nix

environment.systemPackages = with pkgs; [
  proselint
];

From source

git clone https://github.com/amperser/proselint.git
cd proselint

uv pip install . # or, with Nix
nix profile install .

Configuration

The default configuration is:

{
    "max_errors": 1000,
    "checks": {
        "annotations": true,
        "archaism": true,
        "cliches": true,
        "dates_times": true,
        "hedging": true,
        "industrial_language": true,
        "lexical_illusions": true,
        "malapropisms": true,
        "misc": true,
        "mixed_metaphors": true,
        "mondegreens": true,
        "needless_variants": true,
        "nonwords": true,
        "oxymorons": true,
        "psychology": true,
        "redundancy": true,
        "restricted": false,
        "skunked_terms": true,
        "social_awareness": true,
        "spelling": true,
        "terms": true,
        "typography": true,
        "uncomparables": true,
        "weasel_words": true
    }
}

Proselint searches for configuration files in the following order of priority:

  • .proselintrc.json in the current working directory
  • $XDG_CONFIG_HOME/proselint/config.json
  • $HOME/.proselintrc.json
  • /etc/proselintrc (system-wide default)

Thus, if you want to override any rule locally, all you'd need to do is create a .proselintrc.json and set the rule path to false. For example, to disable typography.symbols.curly_quotes: 

{
    "checks.typography.symbols.curly_quotes": false
}
