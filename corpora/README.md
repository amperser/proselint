# Corpora

We curated a corpus of writing to use in testing.

Here, we give the references for all the texts in our "clean" corpus, which
contains pieces from vetted and edited magazines; we expect the linter to run
nearly cleanly over this corpus, which we occasionally verify by hand.

+ [Politics and the English Language] by [George Orwell]
+ [Men I'm Not Married To] by [Dorothy Parker]
+ [Eminent Victorians] by [Lytton Strachey]
+ [A Hunger Artist] by [Franz Kafka]
+ [De Profundis] by [Oscar Wilde]
+ [Climate and health at a critical juncture] by [PLOS]
+ [Publishing with purpose: Bringing lived experiences into journal
  development] by [PLOS]
